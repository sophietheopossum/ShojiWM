# Installation

:::note
This page is a placeholder. The exact build steps will be documented later.
:::

## Prerequisites

- A Linux system with a working Wayland/DRM setup
- A recent Rust toolchain
- Node.js 18 or newer

## Build

```bash
git clone https://github.com/bea4dev/ShojiWM.git
cd ShojiWM
cargo build --release
```

## Run

```bash
./target/release/shoji_wm
```
