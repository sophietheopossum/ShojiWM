# M5-T1: Style System Expansion

## Goal

MVP を超えて実用的な style object を拡張する。

## Scope

- typography
- border variants
- opacity
- alignment
- visibility

## Deliverables

- style prop の拡張
- Rust 側 primitive への反映

## Done When

- 主要 style が type-safe に書ける
- 典型的 SSD 表現に困らない

## Notes

この時点でも full CSS は不要。
