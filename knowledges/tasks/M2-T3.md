# M2-T3: Core TS Components

## Goal

SSD DSL の最小 component 群を TypeScript 側で定義する。

## Scope

- `Box`
- `Label`
- `Button`
- `Window`
- `WindowBorder`
- `AppIcon`

## Deliverables

- typed props 定義
- TSX から使える component API

## Done When

- `WINDOW_MANAGER.decoration = (...) => (...)` の形が書ける
- `Window` slot を含む tree を TS で返せる

## Notes

component は DOM を返さず node descriptor を返す。
