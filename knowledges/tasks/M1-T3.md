# M1-T3: Minimal SSD Rendering

## Goal

Rust 側だけで最小 SSD を描画できるようにする。

## Scope

- border rect
- titlebar background
- label text
- client surface slot

## Deliverables

- render primitive 定義
- Smithay render element への変換
- focus 状態で border color を切り替える最小例

## Done When

- Rust だけでタイトルバーつき window が描ける
- focused / unfocused で見た目が変わる
- client surface が titlebar の下に正しく配置される

## Notes

まずは単色矩形ベースでよい。shadow や blur は不要。
