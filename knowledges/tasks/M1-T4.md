# M1-T4: SSD Hit-Test And Actions

## Goal

SSD 上のクリックを move/resize/button action に変換できるようにする。

## Scope

- titlebar drag
- border resize hit-test
- close button click

## Deliverables

- `HitTestResult`
- `WindowAction`
- pointer event から action 実行までの接続

## Done When

- titlebar をドラッグすると move できる
- border 端で resize が始まる
- close button で window を閉じられる

## Notes

hover/active state は M3 以降でもよい。
