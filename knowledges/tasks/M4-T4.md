# M4-T4: Node Runtime Bootstrap

## Goal

TypeScript/TSX SSD DSL を実行する Node runtime を立ち上げる。

## Scope

- Node 実行基盤
- TS/TSX loader
- `packages/shoji_wm` を runtime から解決できること
- minimal config entrypoint

## Deliverables

- Node ベースの decoration evaluator 起動経路
- TSX config を実行できる最小 launcher

## Done When

- TSX で書いた decoration function を Node で評価できる
- user が明示的に `tsc` を叩かなくても動く

## Notes

最初は `tsx` 等を使ってよい。最適化は不要。
