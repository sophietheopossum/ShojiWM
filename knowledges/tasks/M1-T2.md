# M1-T2: Minimal SSD Layout Engine

## Goal

Rust 側だけで SSD 用の最小 layout を計算できるようにする。

## Scope

- `Box` row/column
- fixed size
- flex grow
- padding
- gap
- border thickness を layout に反映

## Deliverables

- layout tree -> geometry の計算
- `Window` slot の表示領域決定

## Done When

- titlebar + client surface の縦レイアウトが計算できる
- close button の表示領域が確定する
- border 分を含んだ outer geometry を求められる

## Notes

フル flexbox は不要。SSD に必要な subset のみ。
