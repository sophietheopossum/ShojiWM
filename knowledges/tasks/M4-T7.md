# M4-T7: Manual End-to-End SSD Test

## Goal

TypeScript SSD が実際に ShojiWM 上で動くことを手動統合テストで確認する。

## Scope

- kitty
- dolphin
- focused/unfocused
- close button
- titlebar drag
- border resize

## Deliverables

- 手動テスト手順
- 期待結果
- 不具合の洗い出し

## Done When

- 少なくとも 2 種類のアプリで TSX SSD が表示される
- focused 色変化と close button が動作する
- 主要な入力挙動が確認できる

## Notes

ここで初めて「本当に TS で SSD を書いて動かせた」と言える。
