# M2-T4: Rust-TS AST Bridge

## Goal

TypeScript 側の node tree を Rust 側の `DecorationTree` に変換できるようにする。

## Scope

- serialization format
- schema validation
- Rust 側デコード

## Deliverables

- AST 受け渡し仕様
- validation error の出し方

## Done When

- TSX で書いた decoration tree を Rust 側で読める
- invalid tree を Rust 側で拒否できる

## Notes

最初は JSON-like plain object でよい。
