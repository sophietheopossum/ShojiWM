# M2-T2: Custom JSX Runtime

## Goal

`shoji_wm/jsx-runtime` を実装し、TSX から独自 AST を返せるようにする。

## Scope

- `jsx`
- `jsxs`
- `Fragment`

## Deliverables

- JSX runtime 実装
- TSX node の plain object 表現

## Done When

- `<Box><Label /></Box>` が独自 node tree に変換される
- React DOM なしで TSX が動く

## Notes

最初は dev/prod runtime 分離不要。
