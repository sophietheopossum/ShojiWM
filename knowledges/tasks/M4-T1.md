# M4-T1: Minimal Signal System

## Goal

AGS 的な最小 signal system を導入する。

## Scope

- `signal`
- `computed`
- `effect`

## Deliverables

- runtime signal API
- TS 側評価モデルへの接続

## Done When

- signal 更新で decoration の再評価が走る
- focused state を reactive に書ける

## Notes

外部依存を使うなら `@preact/signals-core` が第一候補。
