# M3-T2: Dynamic Decoration Evaluation

## Goal

window ごとの状態に応じて decoration tree を動的に返せるようにする。

## Scope

- focused で色を変える
- XWayland / Wayland 分岐
- title / icon 反映

## Deliverables

- `WINDOW_MANAGER.decoration(window)` の実行経路
- per-window reevaluation のトリガ

## Done When

- window 状態が変わると decoration も更新される
- focus color change が動く

## Notes

最初は full recompute でもよい。
