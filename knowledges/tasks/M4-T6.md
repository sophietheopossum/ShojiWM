# M4-T6: Compositor Render And Input Integration

## Goal

Node から返った decoration tree を実際の ShojiWM 描画経路と hit-test 経路に接続する。

## Scope

- decode 後の `DecorationTree` を layout/render に接続
- client surface と SSD の合成
- hit-test 結果を move/resize/button action に接続

## Deliverables

- 1 window に対して TSX SSD が表示される
- titlebar drag / close button / border resize が動く

## Done When

- ShojiWM 上で TSX SSD が実際に見える
- input も含めて SSD として成立している

## Notes

最初は 1 window / full recompute でもよい。
