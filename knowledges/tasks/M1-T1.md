# M1-T1: SSD Data Model Definition

## Goal

Rust 側で SSD を表現する最小データモデルを定義する。

## Scope

- `DecorationTree`
- node kind 定義
- style の最小表現
- `Window` slot の特別扱い

## Deliverables

- Rust の AST 型定義
- node validation の入口
- `Window` slot が 0 個または複数個のときに弾ける設計

## Done When

- decoration tree を Rust 側で構築できる
- layout/render 実装がこの tree を入力に取れる
- `Window` slot の制約が明文化されている

## Notes

この段階では TS 側との bridge はまだ不要。
