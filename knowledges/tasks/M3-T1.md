# M3-T1: Window Model For TS

## Goal

TypeScript から参照する `WaylandWindow` API を定義する。

## Scope

- title
- app id
- icon
- focused
- floating/maximized/fullscreen
- close/maximize 系 action

## Deliverables

- `WaylandWindow` 型
- Rust 側 window model との対応

## Done When

- TS 側で `window.title`, `window.isFocused`, `window.close()` が使える

## Notes

最初は読み取り専用中心でよい。
