# M5-T2: `css={...}` Support Strategy

## Goal

文字列 CSS 風 API を導入するか、その場合どこまで対応するかを決める。

## Scope

- 文字列 CSS を parse するか
- object style の sugar に留めるか
- サポートする property subset

## Deliverables

- `css` prop の仕様
- 対応プロパティ一覧

## Done When

- `style` と `css` の役割分担が決まっている
- 文字列ベースを導入するかどうかの判断が固まる

## Notes

最初は `style` のみでも十分。`css` は後追いでよい。
