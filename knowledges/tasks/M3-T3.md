# M3-T3: Interaction States In DSL

## Goal

TSX 側で hover / active / focused を使った見た目分岐ができるようにする。

## Scope

- button hover
- button pressed
- root focused

## Deliverables

- component state model
- style への反映

## Done When

- close button の hover/pressed 表現が変わる
- border/titlebar の focused state が TS 側で書ける

## Notes

hover/active は compositor が authoritative に持つ。
