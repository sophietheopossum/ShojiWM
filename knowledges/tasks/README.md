# SSD Task Breakdown

`docs/typescript-ssd-plan.md` のマイルストーンを、細かすぎないタスク単位に分解したもの。

命名規則:

- `M1-T1.md` = Milestone 1, Task 1
- `M2-T3.md` = Milestone 2, Task 3

推奨の進め方:

1. M1 を完了して Rust 側だけで最小 SSD を成立させる
2. M2 で TypeScript runtime/AST bridge を追加する
3. M3 で window 情報依存の dynamic SSD を動かす
4. M4 で signals を導入する
5. M5 で style system を拡張する
