# M3-T4: Practical Default SSD Theme

## Goal

実用的な標準 SSD テーマを 1 つ TSX で書く。

## Scope

- titlebar
- icon
- close button
- focused/unfocused color

## Deliverables

- demo 用 TSX decoration
- スクリーンショットが撮れるレベルの見た目

## Done When

- kitty / dolphin 等で標準 SSD が見える
- 実際の利用例として plan の例コードに近いものが動く

## Notes
