# M2-T1: TypeScript Package Bootstrap

## Goal

`packages/shoji_wm` を workspace package として作成する。

## Scope

- `package.json`
- `tsconfig.json`
- `src/index.ts`
- export 方針

## Deliverables

- workspace から import できる package
- `shoji_wm` という import 名を固定

## Done When

- `import { Box } from "shoji_wm"` が通る
- local workspace package として参照できる

## Notes

`node_modules` 直接編集はしない。
