# M4-T5: TS Decoration Evaluation Integration

## Goal

Node runtime が `WINDOW_MANAGER.decoration(window)` を評価し、Rust 側へ AST を返せるようにする。

## Scope

- window snapshot を Node 側へ渡す
- TSX tree を serialize する
- Rust 側で decode する

## Deliverables

- Rust -> Node -> TSX -> JSON AST -> Rust の往復経路
- per-window evaluation API

## Done When

- `WaylandWindowSnapshot` を入力に decoration AST が返る
- focused / unfocused で結果が変わることを確認できる

## Notes

最初は同期的でもよい。キャッシュやインクリメンタル最適化は後回し。
