# M4-T2: Reactive Window Bindings

## Goal

window 情報を signal 的に TS 側へ流せるようにする。

## Scope

- title
- focus
- app id
- maximized/fullscreen

## Deliverables

- reactive `WaylandWindow`
- update propagation

## Done When

- `window.isFocused` に応じて UI が自動更新される
- title 変更で label が更新される

## Notes

まずは coarse-grained update でもよい。
